{% if lang == 'es' %} Los pagos pueden procesarse mediante proveedores externos, incluyendo Stripe. {% if lang == 'es' %} Las suscripciones de pago pueden renovarse automáticamente al final de cada período de facturación, salvo cancelación previa a la próxima fecha de renovación. {% else %} Paid subscriptions may renew automatically at the end of each billing period unless cancelled before the next renewal date. {% endif %}

{% if lang == 'es' %} Al suscribirte a un plan de pago autorizas cobros recurrentes conforme al plan seleccionado hasta que la suscripción se cancele. {% else %} By subscribing to a paid plan, you authorize recurring charges consistent with the selected plan until the subscription is cancelled. {% endif %}

{% if lang == 'es' %}4. Cancelación{% else %}4. Cancellation{% endif %}

{% if lang == 'es' %} Puedes cancelar tu suscripción antes de la siguiente renovación. {% if lang == 'es' %} Cuando corresponda, el acceso a funciones de pago podrá continuar hasta el final del período ya pagado. {% else %} Where applicable, access to paid features may continue until the end of the already-paid billing period. {% endif %}

{% if lang == 'es' %} Salvo que la ley aplicable exija otra cosa, los pagos de suscripción generalmente no son reembolsables una vez procesados. {% else %} Unless required by applicable law, subscription payments are generally non-refundable once processed. {% endif %}

{% if lang == 'es' %} BillBeam puede considerar solicitudes de reembolso caso por caso, por ejemplo en situaciones de cobros duplicados, errores técnicos claros o circunstancias excepcionales. {% if lang == 'es' %} BillBeam puede actualizar precios, límites o funciones de planes.
