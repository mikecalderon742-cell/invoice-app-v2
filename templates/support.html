{% if lang == 'es' %} No puedo iniciar sesión {% else %} I cannot log in {% endif %}

{% if lang == 'es' %} Verifica que estás usando la dirección correcta, revisa errores de escritura y prueba el flujo de restablecimiento de contraseña. Si sigues sin poder acceder, contáctanos desde el correo asociado a tu cuenta. {% else %} Confirm that you are using the correct email address, check for typing mistakes, and try the password reset flow. If you still cannot access your account, contact us from the email associated with it. {% endif %}

{% if lang == 'es' %} Mi factura o PDF no se ve bien {% else %} My invoice or PDF does not look right {% endif %}

{% if lang == 'es' %} Incluye el número de factura, una captura de pantalla del problema y una breve explicación de lo que esperabas ver. También ayuda indicar si el problema ocurre en la vista pública, en la vista interna o en ambos lugares. {% else %} Include the invoice number, a screenshot of the issue, and a short explanation of what you expected to see. It also helps to mention whether the issue appears in the public view, the internal view, or both. {% endif %}

{% if lang == 'es' %} El correo de factura no se envió {% else %} The invoice email did not send {% endif %}

{% if lang == 'es' %} Comparte el número de factura, la dirección de destino y cualquier mensaje de error visible. Si el PDF se descargó correctamente pero el correo falló, indícanos eso también. {% else %} Share the invoice number, the destination email address, and any visible error message. If the PDF downloaded correctly but the email failed, mention that too. {% endif %}

{% if lang == 'es' %} El enlace público o el pago no funciona {% else %} The public link or payment flow is not working {% endif %}

{% if lang == 'es' %} Envía el número de factura, el enlace si lo tienes, la hora aproximada en que ocurrió el problema y el comportamiento observado. Si el pago fue intentado, incluye cualquier mensaje mostrado por la pasarela. {% else %} Send the invoice number, the link if you have it, the approximate time the issue occurred, and what behavior you observed. If a payment was attempted, include any message shown by the payment flow. {% endif %}

{% if lang == 'es' %} Qué incluir en tu mensaje {% else %} What to include in your message {% endif %}

  • {% if lang == 'es' %} El correo asociado a tu cuenta. {% else %} The email associated with your account. {% endif %}
  • {% if lang == 'es' %} El número de factura, si aplica. {% else %} The invoice number, if applicable. {% endif %}
  • {% if lang == 'es' %} El navegador y dispositivo que estabas usando. {% else %} The browser and device you were using. {% endif %}
  • {% if lang == 'es' %} Capturas de pantalla o el texto exacto del error. {% else %} Screenshots or the exact error text. {% endif %}
  • {% if lang == 'es' %} Una explicación breve de lo que pasó y lo que esperabas que pasara. {% else %} A brief explanation of what happened and what you expected to happen. {% endif %}
