{% if lang == 'es' %} ¿Necesito tarjeta para empezar? {% else %} Do I need a credit card to get started? {% endif %}

{% if lang == 'es' %} No. El plan Starter de BillBeam te permite crear hasta 10 facturas al mes sin introducir tarjeta. {% if lang == 'es' %} ¿Cuál es la diferencia entre Starter y Pro? {% else %} What’s the difference between Starter and Pro? {% endif %}

{% if lang == 'es' %} Starter está pensado para freelancers y negocios pequeños que emiten pocas facturas al mes. Pro está pensado para cuando facturar se vuelve una parte constante de tu semana. {% else %} Starter is designed for freelancers and small businesses who send only a few invoices each month. Pro is for when billing becomes a regular part of your week. {% endif %}

    {% if lang == 'es' %}
  • Starter: hasta 10 facturas al mes y panel básico.
  • Pro: facturas ilimitadas, plantillas adicionales, envío por email, enlaces públicos y más automatización.
    • {% else %}
  • Starter: up to 10 invoices per month and a basic dashboard.
  • Pro: unlimited invoices, extra templates, built-in email sending, public links, and more automation.
    • {% endif %}

{% if lang == 'es' %} ¿Puedo enviar facturas por correo desde BillBeam? {% else %} Can I send invoices by email directly from BillBeam? {% endif %}

{% if lang == 'es' %} Sí. En el plan Pro puedes enviar facturas por correo con el PDF adjunto y un enlace público para tu cliente. En Starter siempre puedes descargar el PDF y enviarlo manualmente desde tu correo. {% else %} Yes. On the Pro plan you can send invoices by email with the PDF attached and a public link for your client. On Starter, you can always download the PDF and send it manually from your own email. {% endif %}

{% if lang == 'es' %} ¿Qué pasa con mis datos si dejo de usar BillBeam? {% else %} What happens to my data if I stop using BillBeam? {% endif %}

{% if lang == 'es' %} Tus facturas y datos de clientes siguen siendo tuyos. Puedes exportar PDFs y volver a tu sistema anterior si lo necesitas. Si en algún momento quieres eliminar tus datos, puedes escribirnos y los eliminaremos de forma segura. {% else %} Your invoices and client data remain yours. You can export PDFs and move back to your previous system if needed. If you ever want your data removed, you can contact us and we will delete it safely. {% endif %}

{% if lang == 'es' %} ¿BillBeam funciona con clientes en otros países e idiomas? {% else %} Does BillBeam work with clients in other countries and languages? {% endif %}

{% if lang == 'es' %} Sí. La interfaz está disponible en inglés y español, y puedes escribir tus descripciones de servicios en el idioma que necesites. The interface is available in English and Spanish, and you can write service descriptions in whatever language you need.
