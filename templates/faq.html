{% if lang == 'es' %} ¿Necesito tarjeta para empezar? {% else %} Do I need a credit card to get started? {% endif %}

{% if lang == 'es' %} No. El plan Starter de BillBeam te deja crear hasta 10 facturas al mes sin introducir tarjeta. Cuando de verdad te esté ahorrando tiempo y dolores de cabeza, puedes pasar al plan Pro. {% else %} No. BillBeam’s Starter plan lets you create up to 10 invoices per month with no card on file. Once it’s clearly saving you time and stress, you can upgrade to Pro. {% endif %}

{% if lang == 'es' %} ¿Cuál es la diferencia entre Starter y Pro? {% else %} What’s the difference between Starter and Pro? {% endif %}

{% if lang == 'es' %} Starter está pensado para freelancers y negocios pequeños que emiten pocas facturas al mes. Pro está pensado para cuando ya facturas de forma constante. {% else %} Starter is designed for solo freelancers and small businesses who send a few invoices each month. Pro is for when billing becomes a regular part of your week. {% endif %}

    {% if lang == 'es' %}
  • Starter: hasta 10 facturas/mes, panel básico.
  • Pro: facturas ilimitadas, plantillas adicionales, envíos por email, enlaces públicos y más automatización.
    • {% else %}
  • Starter: up to 10 invoices / month, basic dashboard.
  • Pro: unlimited invoices, extra templates, email sending, public links, and more automation.
    • {% endif %}

{% if lang == 'es' %} ¿Puedo enviar facturas por correo electrónico desde BillBeam? {% else %} Can I send invoices by email directly from BillBeam? {% endif %}

{% if lang == 'es' %} Sí, en el plan Pro puedes enviar facturas por email con el PDF adjunto y un enlace público para tu cliente. Desde Starter siempre puedes descargar el PDF y enviarlo manualmente desde tu correo. {% else %} Yes. On the Pro plan you can send invoices by email with the PDF attached and a public link for your client. On Starter, you can always download the PDF and send it manually from your own email. {% endif %}

{% if lang == 'es' %} ¿Qué pasa con mis datos si dejo de usar BillBeam? {% else %} What happens to my data if I stop using BillBeam? {% endif %}

{% if lang == 'es' %} Tus facturas y clientes siguen siendo tuyos. Puedes exportar PDFs y usar tu propio sistema (Excel, CRM, etc.). Si en el futuro quieres borrar tus datos, puedes escribirnos y los eliminaremos de forma segura. {% else %} Your invoices and client data are yours. You can export PDFs and move back to your old system (Excel, CRM, etc.). If you ever want your account wiped, you can reach out and we’ll delete your data safely. {% endif %}

{% if lang == 'es' %} ¿BillBeam es compatible con clientes en otros países e idiomas? {% else %} Does BillBeam work with clients in other countries and languages? {% endif %}

{% if lang == 'es' %} Sí. La interfaz está disponible en inglés y español, y puedes escribir la descripción de tus servicios en el idioma que necesites. También puedes ajustar moneda, impuestos y notas en tus plantillas. {% else %} Yes. The interface is available in English and Spanish, and you can write invoice descriptions in whatever language you need. You can also adjust currency, tax details, and notes in your templates. {% endif %}

{% if lang == 'es' %} ¿BillBeam reemplaza a mi contable? {% else %} Does BillBeam replace my accountant? {% endif %}

{% if lang == 'es' %} No. BillBeam te ayuda con el día a día de emitir y organizar facturas, pero no da asesoría legal, fiscal o contable. Para temas de impuestos o normativa siempre es mejor hablar con un profesional. {% else %} No. BillBeam helps with the day-to-day of sending and organizing invoices, but it doesn’t give legal, tax, or accounting advice. For tax questions or compliance, it’s always best to talk to a professional. {% endif %}
